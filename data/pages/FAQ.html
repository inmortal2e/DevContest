FAQ

  1. Jaké programovací jazyky systém podporuje

    V současnosti je nasazen a plně ozkoušen kompilátor pro C, Pascal a Javu. Plně podporované nekompilované jazyky jsou Python a Ruby. Můžete odesílat řešení také v C++, PHP, za jejich spolehlivost ale neručíme (jako ostatně za nic, že? :) ). Pokud by byl zájem, není problém přidat podporu např. Haskellu, Lispu apod.

  2. Jaké jsou příslušné přípony pro daný jazyk?

    C *.c
    Pascal *.pas
    Python *.py
    Java *.java
    Ruby *.rb
    C++ *.cpp
    PHP *.php

  3. Jsou nějak programovací jazyky omezeny?

    Jsou i nejsou. Není povoleno jakkoli manipulovat se soubory na disku, ale to je asi veškeré bezpečnostní opatření. Až přijdete na bezpečnostní díru, nesmažte nám server, ale raději nám pomožte to opravit. Dostanete čokoládu :)

  4. Musí být soubory pojmenovány podle úlohy?

    Nemusí. Robot si stejně soubor přejmenuje podle svého úsudku. Nedoporučujeme používat v názvech souborů diakritiku a mezery.

  5. Musím celou soutěž řešit pomocí jednoho jazyka?

    Nemusíte. Časem možná toto omezení přibude, zatím není naimplementováno.

  6. Jak mám načítat vstup a jak vypisovat výstup?

    Vstup se načítá ze standardního vstupu (klávesnice) a výstup se vypisuje na standardní výstup (obrazovka). Pro usnadnění ladění programu je vhodné použít přesměrování V/V souborů.

  7. Může program vypisovat výstup ještě před načtením celého vstupu?

    Ano, program může vypisovat výstup dříve než dočte celý vstup. Jinými slovy, správnost výstupu programu se kontroluje až po jeho skončení.

  8. Je potřebné kontrolovat správnost vstupu?

    Ne, programy jsou testované jen na vstupech, které vyhovují zadání.

  9. Mohli byste můj program ručně zkontrolovat?

    Jen když si budete sakramentsky jistí, že máte program správně a přesto vám nefunguje. Jinak naštvete organizátora a čert ví co vám může provést.

  10. Kontroluje se časová náročnost programů?

    Musí proběhnout do jedné sekundy. Vstupy jsou dělány tak, že narychlo splácané autorské řešení to nemá problém zvládnout.
